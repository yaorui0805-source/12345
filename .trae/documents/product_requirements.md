# AI分身创建页面 - 产品需求文档

## 产品概述
重新设计AI分身创建页面，废弃传统的长文本表单输入方式，采用更直观、交互性更强的滑块选择+标签选择+深度问答的方式来帮助用户创建个性化的AI分身。

## 核心功能变更

### 1. 人设设置模块（重构）
**废弃功能：**
- 长文本输入框让用户手动输入人设描述
- 声音设置模块
- 虚拟形象设置模块

**新增功能：**

#### 1.1 性格维度滑块
用户通过滑块选择AI分身的性格倾向：
- 外向 ←→ 内向 (Extraversion vs Introversion)
- 感性 ←→ 理性 (Emotional vs Rational)
- 活泼 ←→ 沉稳 (Energetic vs Calm)
- 乐观 ←→ 悲观 (Optimistic vs Pessimistic)
- 随和 ←→ 严谨 (Easygoing vs Serious)

#### 1.2 语言风格滑块
用户通过滑块选择AI分身的语言风格：
- 夸夸 ←→ 锐评 (Praise vs Critical)
- 抽象 ←→ 客观 (Abstract vs Objective)
- 幽默 ←→ 正经 (Humorous vs Formal)
- 温柔 ←→ 直接 (Gentle vs Direct)
- 文艺 ←→ 通俗 (Literary vs Casual)

#### 1.3 个人标签Tag选择
预设热门标签供用户选择，支持多选：
- 游戏类：王者、吃鸡、原神、Steam、Switch
- 娱乐类：追星、追剧、综艺、电影、音乐
- 生活类：美食、旅行、摄影、健身、萌宠
- 文化类：二次元、古风、嘻哈、摇滚、国风
- 地域类：川渝、东北、广东、江浙沪、北京
- 其他：社恐、社牛、打工人、学生党、熬夜冠军

#### 1.4 抖音一键导入（保留）
- 保留"还原抖音人设"功能按钮
- 点击后自动从抖音账号提取人设信息
- 自动填充滑块位置和标签选择

### 2. 深度灵魂问答（新增）
**核心流程：**
1. AI分身（无形象）主动向用户发起多轮对话
2. 每轮提出一个深度问题，了解用户的价值观、兴趣爱好、生活态度等
3. 根据用户回答自动分析并优化人设参数
4. 将关键信息录入AI分身的长期记忆

**问答示例：**
- "如果你能拥有一种超能力，你会选择什么？为什么？"
- "描述一下你理想中的周末是怎样的？"
- "你最看重朋友身上的什么品质？"
- "如果可以给五年前的自己一条建议，你会说什么？"
- "你最近一直在思考的一个问题是什么？"

**记忆录入：**
- 自动提取回答中的关键信息
- 分类存储：兴趣爱好、价值观、生活态度、人际关系等
- 支持用户手动编辑和补充记忆

## 交互设计

### 滑块交互
- 滑块范围：0-100
- 默认位置：50（中间平衡状态）
- 滑动时实时显示当前偏向
- 支持点击轨道快速定位

### 标签交互
- 点击选中/取消选中
- 选中的标签显示高亮样式
- 支持横向滑动查看更多标签
- 已选标签显示在顶部汇总区域

### 问答交互
- 对话式界面，类似聊天应用
- AI提问以气泡形式展示
- 用户输入框固定在底部
- 支持语音输入（可选）
- 回答后AI给出反馈并继续提问
- 可随时结束问答进入下一步

## 技术实现要点

### 状态管理
- 使用Zustand管理表单状态
- 滑块值、标签选择、问答历史统一存储
- 支持草稿自动保存

### AI问答实现
- 调用大模型API生成个性化问题
- 根据用户回答实时分析性格特征
- 自动调整滑块建议和标签推荐

### 数据模型
```typescript
interface AIAvatarConfig {
  personality: {
    extraversion: number;      // 外向-内向 0-100
    rationality: number;       // 感性-理性 0-100
    energy: number;            // 活泼-沉稳 0-100
    optimism: number;          // 乐观-悲观 0-100
    seriousness: number;       // 随和-严谨 0-100
  };
  languageStyle: {
    praiseCritical: number;    // 夸夸-锐评 0-100
    abstractObjective: number; // 抽象-客观 0-100
    humorousFormal: number;    // 幽默-正经 0-100
    gentleDirect: number;      // 温柔-直接 0-100
    literaryCasual: number;    // 文艺-通俗 0-100
  };
  tags: string[];              // 选中的标签
  memories: MemoryItem[];      // 问答提取的记忆
  douyinImported: boolean;     // 是否从抖音导入
}

interface MemoryItem {
  id: string;
  category: 'interest' | 'value' | 'attitude' | 'relationship';
  content: string;
  source: 'qa' | 'manual';
  createdAt: number;
}
```
